# Telegram Channel Autopilot Bot

Готовый Python-проект для Telegram-канала:
- постит **2 раза в день** по расписанию;
- берет контент из разных источников: новости, технологии, игры, праздники, космос, фото дня;
- удаляет комментарии с матом и оскорблениями в привязанной группе обсуждений;
- предупреждает пользователей, что мат и оскорбления запрещены;
- собирает метрики по постам и постепенно отдает приоритет категориям, которые заходят лучше;
- умеет вручную публиковать пост и смотреть топ постов через команды владельца.

## Важно про токен
Токен бота, который вы прислали в чате, уже считается скомпрометированным. Его лучше **сразу перевыпустить через @BotFather** и использовать новый токен только в `.env`. Telegram прямо рекомендует хранить токен в безопасном месте и не публиковать его.

## Что реально умеет Telegram
Бот API позволяет ботам публиковать посты, работать через polling/webhook и получать обновления. Актуальная версия Bot API на сегодня — 9.5.

По аналитике есть ограничение: **через обычный Bot API нельзя полноценно получить всю статистику канала как в интерфейсе Telegram**. Для этого в проекте добавлен **опциональный модуль Telethon**, который может читать просмотры, количество реакций и комментариев у опубликованных сообщений, если вы настроите `TELEGRAM_API_ID` и `TELEGRAM_API_HASH` от своего аккаунта Telegram API. У крупных каналов расширенная статистика вообще относится к отдельному API статистики каналов.

## Структура
- `app/main.py` — запуск бота
- `app/data/sources.yaml` — источники контента
- `app/handlers/moderation.py` — удаление плохих комментариев
- `app/handlers/admin.py` — команды владельца
- `app/services/content_sources.py` — сбор контента
- `app/services/analytics.py` — метрики через Telethon
- `app/services/ranking.py` — выбор лучшей категории
- `.env.example` — переменные окружения

## Быстрый запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Потом заполните `.env`:
```env
BOT_TOKEN=НОВЫЙ_ТОКЕН_ОТ_BOTFATHER
CHANNEL_ID=@tgbots_miniapps
DISCUSSION_CHAT_ID=-100XXXXXXXXXX
OWNER_IDS=ВАШ_TELEGRAM_ID
TIMEZONE=Europe/Vienna
POST_TIMES=10:00,19:00
ENABLE_TELETHON_ANALYTICS=false
```

Запуск:
```bash
python -m app.main
```

## Что нужно настроить в Telegram
1. Добавьте бота в канал **админом** с правами на публикацию и закрепление.
2. Если хотите модерацию комментариев — привяжите к каналу группу обсуждений и добавьте туда бота **админом** с правом удалять сообщения.
3. Узнайте `DISCUSSION_CHAT_ID` группы. Проще всего временно вывести `message.chat.id` любым тестовым хендлером или через `getUpdates`.
4. Для расширенной аналитики создайте `api_id` и `api_hash` в Telegram API и включите `ENABLE_TELETHON_ANALYTICS=true`.

## Команды владельца
- `/post_now` — немедленно опубликовать пост
- `/top_posts` — показать лучшие посты по score
- `/sync_stats` — вручную обновить метрики

## Как бот принимает решения
Бот хранит историю постов в `bot.db`. После публикации модуль аналитики обновляет:
- `views`
- `reactions`
- `comments`
- общий `score = views*0.03 + reactions*2 + comments*1.5`

Дальше ранкер повышает шанс тех категорий, у которых средний score выше. Это не магия роста канала, но это помогает постепенно смещать контент в сторону того, что лучше заходит аудитории.
